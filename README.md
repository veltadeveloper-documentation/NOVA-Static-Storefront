# NØVA Static Storefront

Frontend-only premium store preview.

## Structure

- `index.html`: main page shell and semantic sections
- `assets/css/styles.css`: all layout, theme, and animation styles
- `assets/js/data.js`: content data for categories, products, arrivals, and testimonials
- `assets/js/app.js`: rendering, filters, cart-preview interactions, toast messages, and mobile navigation

## Notes

- This version has no backend or checkout integration.
- Cart, search, account, and newsletter actions are demo interactions only.
- The project is designed to work as a clean preview site.
